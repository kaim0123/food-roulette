## Chrome DevTools の使い方（初心者向け）

1. ブラウザで F12 または右クリック →「検証」
2. 左上のスマホアイコン（Device Toolbar）をクリック
3. 上部のドロップダウンでデバイスを選択
4. 画面幅を変えながら表示を確認

## よくある問題と対処

| 問題 | 原因 | 対処 |
|------|------|------|
| フッターと本文が重なる | `pb-` の余白が足りない | `pb-[calc(3.5rem+env(safe-area-inset-bottom))]` を確認 |
| チップが折り返されない | `flex-wrap` がない | `flex flex-wrap` を追加 |
| スロットが画面外にはみ出る | `overflow-hidden` がない | リール親要素に `overflow-hidden` を追加 |
| ボタンが小さすぎる | タッチターゲットが44px未満 | `min-h-[3rem]` 以上を確保 |
