# 模写タスク一覧 - Food Roulette

このドキュメントは、food-roulette アプリを模写（ゼロから再構築）するための
初心者向けタスク一覧です。番号順に進めることで、段階的にアプリが完成します。

## 前提知識

- HTML / CSS の基本
- JavaScript（ES6+）の基本
- React の基本（useState, useEffect）
- ターミナル操作の基本

## 推奨する進め方

1. まず元のアプリを `npm run dev` で動かして、挙動を確認する
2. 別ディレクトリにプロジェクトを作り、各Issueの順に実装する
3. 1つ完了するごとに動作確認 → コミットする
